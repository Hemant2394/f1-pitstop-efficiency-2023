# ⚡ F1 Pit Stop Efficiency Dashboard (2023)

## Overview

This project evaluates pit stop efficiency across Formula 1 constructors during the 2023 season. Using boxplot analysis, it compares the consistency, variability, and operational performance of different pit crews.
